## Readme

This is a sinatra app

### Running the app

From the root of the app type ```ruby app.rb```
Go to `http://localhost:4567/`

### Running test

From the root of the app type ```rspec .``` or ```rspec spec/app_spec.rb```

```curl http://localhost:4567/api/v1/list```
